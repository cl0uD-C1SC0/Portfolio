Crie dois targets groups para o Traefik

NOTE:
    - Traefik listen on two ports: 80 e 8080
    - Port 80 = Target services, the containers that traefik will route to.
    - Port 8080 is for the Admin dashboard and healthcheck /ping

TASK:
    Crie um Target group port 80
    Selecione: IP Addresses
    Protocol: HTTP 80
    VPC: VPc do ECS
    HEALTH CHECKS:
        HTTP
        /ping
        +Advanced health check settings
            Mark: Override
            Port 8080
            Success codes: 200,301,302

    Crie um Target group port 8080
    Selecione: IP Addresses
    Protocol: HTTP 8080
    VPC: VPC do ECS (Ou algo como admin)
    Health checks:
        HTTP
        /ping
        +Advanced health check settings
            Mark: Override
            Port 8080
            Success codes: 200,301,302
