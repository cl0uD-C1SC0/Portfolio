TASK:
    Fargate Service
    Crie pela CLI:

    aws ecs create-service --cli-input-json file://./file.json

    Crie um arquivo json:
    (service-template.json)

    OBS: Cuidado no IAM
    OBS: Nome do Cluster
    OBS: Nome do service


TASK 4 para cada Task Definition que o JAM tem.


    - Veja se tem um task definition de cada /api/1-0/*
    - Entre no task definition e procure pelo container
    - Dentro do container procure por Docker Label
    OBS: O Traefik já deve estar rodando para fazer está ultima TASK!

    - Para cada nova task definition crie um Security group e permita regra HTTP 80 e HTTPS 443 do SG do Traefik

    EX:

    SG hamilton:
        HTTP 80 > SG Traefik (traefik-config.jpg)

    - Em Docker Label digie:
        Key traefik.enable 
        Value true

        Key traefik.http.routers.whoami.rule
        Value Host(`alb-address`) && Path(`/api/1-0/path1`)

        Troque os valores:
                alb-address pelo DNS do ALB
                path1 pelos valores que eles pedem!


OBS: Eu digitei esse comando no CMD, não sei se vai precisar: (Healthcheck command)
    CMD-SHELL /usr/bin/wget --spider http://traefik-691756010.us-east-1.elb.amazonaws.com:8080/ping || exit 1
