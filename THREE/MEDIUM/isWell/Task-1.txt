Se tiver uma EC2 conecte nela 
Ou senão faça pelo CMD do Computador.

CMD:
    AWS Configure (Configure as credenciais normalmente)
    Se os comandos não funcionarem e derem error de session_token execute:
    aws configure set aws_session_token "session_token_do_jam"

Comandos:
    aws kinesis list-shards --stream-name <nome-do-kinesis>
    (Procure pelo ShardID que tem StartingHashKey 0)

    Depois:
    aws kinesis split-shard --stream-name <nome-do-kinesis> --shard-to-split <shard-id> --new-starting-hash-key 10

    Valide.
