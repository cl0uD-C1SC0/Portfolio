# QUERYING WITH ORDER BY:
"organiza os resultados de acordo com uma ou mais colunas da tabela, podendo definir a
ordem do resultados como crescente ou decrescente. Caso a ordem não seja declarada
será crescente (Do menor para o maior)

EX:
	Select * FROM numer ORDER BY number

Ele vai pegar do database numer, tudo e organizar de 1, 2, 3, 4, 5...

	Select * FROM number ORDER BY id ASC (Organiza do a LABEL ID, de 1 para cima)
	Select * FROM number ORDER BY id DESC (Organiza do a LABEL ID, de 10 para baixo ou 20 para baixo EX: 50, 49, 48, 47, 46, 45)

--------------------
AVG:

SELECT "colum_name1,"colum_name2", AVG(expressions) FROM "table_name" GROUP BY "column_name1","column_name2"

OBS:
	colum_name1 e 2 = Nome das colunas (LABEL)
	AVG(expressions) = Pode ser um nome de coluna ou uma operação arithmetic. Uma operação arithmetic pode incluir mais de uma coluna.
	GROUP BY = Organiza como será mostrado as colunas.
